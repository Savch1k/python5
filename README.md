# Генератор цифр числа π

Программа демонстрирует использование генераторов и функции `map` в Python для обработки цифр числа π.

## Описание

Программа выполняет следующие действия:

1. Генерирует цифры числа π (первые N цифр) с помощью функции-генератора
2. Применяет к каждой цифре преобразование: `digit / digit²` (кроме 0)
3. Вычисляет сумму полученных значений
4. Выводит исходные цифры и результат вычислений

## Формула преобразования

Для каждой цифры `d` вычисляется значение:
- `d / d² = 1/d`, если `d ≠ 0`
- `0`, если `d = 0`

Таким образом, программа фактически вычисляет сумму обратных чисел для ненулевых цифр.
